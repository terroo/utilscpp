# ascii2dec
Convert Decimal to ASCII

## Compiling

```sh
mkdir build && cd build
cmake ..
make
```

## Use
> **Use quotes!** 


```sh
./ascii2dec "Welcome to C++ Utils"
# 87 101 108 99 111 109 101 32 116 111 32 67 32 43 43 32 85 116 105 108 115
```
