# Leap Year
Checks one or more years are leap or not

## Compiling

```sh
mkdir build && cd build
cmake ..
make
```

## Use
Run the program entering one or more years as parameter.
```sh
./leap-year 1900 2019 2400 2020
1900 is NOT a leap year.
2019 is NOT a leap year.
2400 is a leap year.
2020 is a leap year.
```
